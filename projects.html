Weather App:

Project Overview: This is a frontend project,a weather app that detects and displays real-time weather conditions worldwide. This project is a clone, and I gave it my best shot on my first attempt.

Technologies Used: HTML, CSS, JavaScript

Features:
Real-Time Weather: Fetches and displays current weather conditions using an API key from a weather API.
Default Location: The app is set to display weather details for Coimbatore by default.
Search Functionality: Users can search for any location, and the app will display the respective weather details. If the location is not found, an error message "City not found" is shown.
User-Friendly Interface: Includes a home page with a few pre-set locations and displays the current date and time, as well as the temperature in Celsius.
Error Handling: If the search bar is left empty, the app prompts with "Please type a city name. "Upon refreshing the page, it reverts to showing the weather conditions of the default location.

Learning Experience: This front-end project allowed me to explore the integration of APIs with JavaScript and improve my skills in HTML and CSS design. It's a foundational step in my journey as a web developer.

Bookstore:

Project Overview: This is a mern Bookstore web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). The project started as a clone of an existing bookstore application, but I made significant modifications to enhance its functionality and design. It allows users to create an account explore the courses and send a message to the store.
Features:
User Authentication: The app includes a login page that allows users to sign in. If a user is new, they are prompted to sign up for an account.
Course Listings: The Courses page displays available courses with attractive images and descriptions, allowing users to explore what the bookstore has to offer.
About Us & Contact Pages: These pages provide details about the store and a contact form where users can send messages directly to the admin. The messages, along with user details, are stored in the MongoDB.
Dark Mode & Light Mode: Users can switch between dark and light themes to suit their preferences.
Customization: Added custom background images and developed the About and Contact pages from scratch to enhance the user experience
Database: The user's login details and the messages sent by the users and their details are stored in the database.The database is connected to mongodb.
Technical Stack:
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB (Used to store user details and contact messages)
